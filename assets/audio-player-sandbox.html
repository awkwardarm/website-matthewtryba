Sandbox Notes

To swap tracks: edit the TRACKS array at the bottom of this file. Each track needs title, artist, src, and optionally artwork. Set src to your Cloudflare R2 public URL once uploaded. The group field controls which section a track appears in — add/remove groups freely.

Audio files: export MP3 320kbps. ~5–8MB each. Upload to Cloudflare R2 bucket with public access enabled. URL format will be: https://pub-XXXX.r2.dev/filename.mp3
